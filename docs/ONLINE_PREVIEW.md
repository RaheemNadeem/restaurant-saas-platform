# Online UI Preview (Headless-friendly)

This repository auto-deploys the Angular frontend preview to GitHub Pages on pushes to `main` affecting `frontend/**`.

## URL
`https://raheemnadeem.github.io/restaurant-saas-platform/`

If deployment is still running, wait 1-3 minutes and refresh.

## Manual trigger
GitHub → Actions → `deploy-pages-preview` → Run workflow.
