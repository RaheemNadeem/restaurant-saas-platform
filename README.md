# Restaurant SaaS Platform

Multi-tenant SaaS platform for restaurants and food businesses.

## Stack
- Frontend: Angular
- Backend: .NET 8 Web API
- Database: PostgreSQL
- Realtime: SignalR
- Payments: Stripe Connect

## Monorepo Structure
- `frontend/` Angular apps/libs
- `backend/` .NET 8 APIs and modules
- `infra/` IaC, deployment, monitoring
- `docs/prds/` Product requirement documents

## Principles
- Scalable multi-tenant architecture
- Security by design
- Modular boundaries
- Production-grade reliability
